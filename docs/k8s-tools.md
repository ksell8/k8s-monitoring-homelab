# Tools, tools, tools

Kubernetes, or k8s, does tons of crazy stuff.  Tools make these crazy things easier to manage.

## kubectl

[kubectl](https://github.com/kubernetes/kubectl) is a CLI interfaces with the kubernetes API server to do stuff.

[Install it.](https://github.com/kubernetes/kubectl)

## k9s

No one has time to remember all the kubectl commands.

[k9s](https://github.com/derailed/k9s) has a puppy mascot, is built in go, and provides a GUI interface with **autocomplete**.

[What are you waiting for?  Install it.](https://k9scli.io/topics/install/)
